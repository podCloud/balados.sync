    authenticate(state, user_id, token_type, token_value, opts \\ [])

    @spec authenticate(t(), String.t(), token_type(), String.t(), keyword()) :: t()

    Authenticates the connection with the given user and token information.

    Transitions from :unauthenticated to :authenticated state.

    Accepts optional device_id and device_name. If not provided, uses defaults ("websocket" and "WebSocket Live").

    authenticated?(state)

    @spec authenticated?(t()) :: boolean()

    Checks if the connection is authenticated.

    check_rate_limit(state)

    @spec check_rate_limit(t()) :: {:ok, t()} | {:error, :rate_limited, t()}

    Checks rate limit and consumes a token if available.

    Returns {:ok, updated_state} if the request is allowed, or {:error, :rate_limited, updated_state} if rate limit exceeded.

    new()

    @spec new() :: t()

    Creates a new connection state.

    The connection starts in the unauthenticated state, ready to receive an auth message. Initializes a rate limiting token bucket for this connection.
