    mix projections.migrate

    Runs migrations for the projection schemas (users, public).

    mix reset_projections

    Resets all projection tables by truncating them and resetting projector subscriptions.

    mix reset_system

    Resets all system tables (users, app_tokens, play_tokens).

    mix system.migrate

    Runs migrations for the system schema (users, app_tokens, play_tokens).

    mix system_db.create

    Creates the system database and initializes the system schema prefix.

    mix system_db.init_schema

    Creates the system schema in the database if it doesn't exist.

    mix system_db.migrate

    Runs migrations for the system schema prefix.
